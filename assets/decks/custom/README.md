# Deck personalizado

Coloque aqui uma versao personalizada das cartas usando os mesmos nomes do deck tradicional:

- `1S.svg` ate `13S.svg` para espadas
- `1H.svg` ate `13H.svg` para copas
- `1D.svg` ate `13D.svg` para ouros
- `1C.svg` ate `13C.svg` para paus

Quando o deck estiver completo, habilite a opcao `Personalizado` em `index.html`.
