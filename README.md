# DRZJ FreeCell

FreeCell estatico para GitHub Pages, feito para desktop e celular em landscape.

## Como publicar

1. Envie os arquivos para o repositorio GitHub.
2. No GitHub, abra `Settings > Pages`.
3. Selecione deploy da branch `main` e pasta `/root`.

## Funcionalidades

- Jogo FreeCell classico com 4 free cells, 4 fundacoes e 8 colunas.
- Timer, recorde local e estado salvo em `localStorage`.
- Desfazer movimentos.
- Auto-promocao opcional.
- Deck tradicional em SVG dentro de `assets/decks/traditional`.
- Estrutura preparada para deck personalizado em `assets/decks/custom`.

Tudo roda no navegador. Nao ha backend, login nem banco de dados.
